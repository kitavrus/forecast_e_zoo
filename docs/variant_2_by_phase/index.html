Вариант 2. Декомпозиция по фазам

Каждая фаза общей диаграммы становится отдельным swimlane. Лейны те же (роли), но каждая ячейка раскрыта в несколько шагов внутри фазы.

Когда удобно

  • Сквозной путь одного цикла «слева направо» виден сразу
  • Понятно, какая роль на каком шаге что делает
  • Согласуется с фазами в README и диаграммах 1..4

Минусы

  • Внутреннее устройство одной роли разбросано по 4 файлам
  • Если фаза почти полностью обслуживается одной ролью — много пустых ячеек
  • Не подсвечивает специфику конкретного потока (нет «зума» в стрелку)
