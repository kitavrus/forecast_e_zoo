Фаза 3. Подготовка заказа — детальная swimlane

Готовый из Ф2 заказ проходит финальную валидацию, обогащается метаданными, маршрутизируется через адаптер провайдеров и передаётся внешним системам поставщика. Параллельно копия идёт в ERP клиента для регистрации в Ф4.
