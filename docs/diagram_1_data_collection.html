Как это работает. ERP E-Zoo — единственный источник истины для всех справочников и фактов. X-Flow раз в сутки выгружает мастер-данные (товары, точки, поставщики, план поставок, промо) и движения (остатки, чеки, перемещения, списания, инвентаризации). Для высоконагруженных потоков используется CDC-обмен — только инкремент изменений. На выходе Replenishment получает пять витрин, готовых для работы модулей прогнозирования и калькуляции.
