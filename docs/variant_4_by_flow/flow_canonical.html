Как читать. Это «зум» в одну подписанную стрелку общей диаграммы — поток canonical между Адаптером источников и X-Flow. Лейны — участники этого конкретного потока, колонки — последовательные стадии превращения сообщения. Ветка staging для непривязанных полей показана пунктиром: оператор разбирает её через UI, после чего маппинг повторно прогоняется. Эта диаграмма — шаблон для остальных flow_*.html.
