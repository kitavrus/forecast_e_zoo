Как читать. Inputs — это «компилятор» сырых витрин в готовое представление для моделей. Промежуточный feature cache позволяет не пересчитывать одно и то же при повторных запросах. Calc-context (остатки, лимиты) идёт по отдельной короткой ветке без feature engineering.
