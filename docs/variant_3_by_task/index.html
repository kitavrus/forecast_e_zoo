Вариант 3. Декомпозиция по карточкам (task-zoom)

Каждая карточка с общей диаграммы (например, «Прогноз → калькуляция → конструктор» или «Маршрутизация каналов») раскрывается в собственный swimlane с подмодулями и шагами.

Когда удобно

  • Максимальная глубина — каждая важная функция в фокусе
  • Хорошо для технического дизайна и оценки скоупа
  • Можно дёрнуть нужную диаграмму прямо из общей по клику

Минусы

  • ~10 файлов — больше всего работы по поддержке
  • Связь между карточками теряется внутри отдельного файла
  • Часть карточек тривиальна (1-2 шага) — диаграмма для них избыточна
